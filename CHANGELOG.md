## [1.0.0] - 2026-04-06

### Added
- Initial release
- `promptchan_generate.py` - Image generation script
- `promptchan_status.py` - Balance and status check script
- `promptchan_styles.py` - List available styles, poses, and filters
- English README (`README.md`)
- Chinese README (`README_zh.md`)
- Japanese README (`README_ja.md`)
- Complete SKILL.md documentation
- GitHub standard files (LICENSE, CONTRIBUTING.md, CODE_OF_CONDUCT.md)
- `.gitignore` for Python projects

### Features
- Support for 15 art styles
- Support for 116 poses
- Support for 11 filters
- Multiple image sizes (512x512, 512x768, 768x512)
- Quality levels (Ultra, Extreme, Max)
- Seed control for reproducible results
- Face restoration option (realistic styles only)
